# Chess Tournament Management System

A Chess Tournament Management System built using Svelte and JavaScript.

## Features

- Player CRUD (Create, Read, Update, Delete)
- Tournament CRUD
- Add players to tournaments
- Random match generation
- Random winner selection
- Final rankings (1st, 2nd, 3rd)

## Tech Stack

- Svelte
- JavaScript
- Vite

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
